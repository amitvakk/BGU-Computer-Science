# Ben-Gurion University - Computer Science Projects

This repository contains key academic projects and laboratory assignments developed during my Computer Science studies at Ben-Gurion University.
It showcases implementation skills in both high-level software design and low-level systems programming.

## Repository Directory

### 📂 [Data-Structures]

* **Project 1: product inventory tracker** A custom Java implementation of advanced data structures designed to manage entities under strict performance requirements. 
* Core Concepts: Self-Balancing AVL Trees, Doubly Linked Lists, and Sorted Arrays.
* Key Achievement: Built entirely from scratch without utilizing Java's built-in collections framework to demonstrate deep algorithmic understanding.

### 📂 [Computer-Architecture-Labs]
Low-level applications written in x86-64 Assembly, focusing on CPU hardware interaction, memory layout, and mathematical algorithms.
* **Project 1: Recursive Logarithm Calculator** – A high-precision math utility utilizing the x87 Floating-Point Unit (FPU) coprocessor and 80-bit extended-precision floats.
* **Project 2: Fraction Calculator** – A command-line utility implementing System V AMD64 ABI compliance, string parsing, and 64-bit signed arithmetic.
